O que so Biblotecas ou Mdulo?

Uma biblioteca uma coleo de mdulos de script acessveis a um programa Python, isto , um pacote de cdigos que est pronto no Python. Dessa forma, voc pode instalar uma biblioteca que foi produzida por outra pessoa e utilizar as ferramentas dessa biblioteca para resolver os problemas que voc est enfrentando.

Bibliotecas Bsicas

Numpy

Definio: uma biblioteca da linguagem Python, chamada de Numerical Python, uma coleo de funes e operaes que ajudam a executar clculos numricos com facilidade. O NumPy oferece uma biblioteca para clculos fceis e rpidos.

Para deixar a ideia mais clara de como funciona a biblioteca, podemos utilizar um exemplo mais prtico. Suponha que voc queira resolver uma equao do segundo grau. Para isto voc precisa, por exemplo, ter um conhecimento das operaes de soma e subtrao. Tendo isso em mente, voc ir atrs de um livro de matemtica bsica para obter tal conhecimento e conseguir executar os clculos. Nesse exemplo, se levarmos para linguagem de programao, podemos dizer que a biblioteca Numpy seria o livro de matemtica bsica, onde se encontra o conhecimento de soma e subtrao. Logo, ao utilizarmos uma funo dessa biblioteca, estamos dizendo para mquina ir nesta biblioteca e pegar um certo "conhecimento" para executar algum clculo.

Pandas

Definio: uma biblioteca da linguagem Python, utilizada para manipulao e anlise de dados. A biblioteca permite ler, manipular, agregar e plotar os dados de forma simples.

Para exemplificarmos esta biblioteca, podemos usar o exemplo da criao de uma matriz. Digamos que voc queira criar uma matriz. Para isso, voc precisa manipular certos dados para encaix-los corretamente na matriz. O Pandas funcionaria como o livro que contm o conhecimento necessrio para manipulao. Logo, ao executar uma funo da biblioteca, estamos pedindo para a mquina acessar este "livro" e executar esses conhecimentos para construir a matriz corretamente.

WebScrapping

Definio: o ato de coletar dados estruturados na Web de maneira automatizada. Dessa forma, podemos chamar o WebScrapping de Raspagem de Dados ou Extrao de Dados da Web - ambas definies refletem bem o que feito pelo WebScrapping. Neste sentido, a Raspagem de Dados desempenha um papel fundamental ao ceder os dados que sero utilizados pelas bibliotecas Numpy ou Pandas e para outros fins.

A prtica de Webscrapping, uma maneira de automatizar o processo da coleta de dados, em uma certa pgina para uma anlise posterior. Para exemplificar este caso, suponha que queremos saber quantas ofertas do produto Macbook air existem na pgina principal do Mercado Livre. Podemos fazer isto manualmente, acessando a pgina e contando um a um. Porm, se quisermos economizar mais tempo e obter uma resposta com uma menor margem de erro, podemos utilizar algumas sequncias de cdigos que basicamente diz para mquina acessar o site e realizar esta contagem. Fazendo isso, utilizamos a capacidade de processamento da mquina para economizar tempo.

Bibliotecas de WebScrapping

urlib

Definio: uma biblioteca para acessar, ler e fazer o parse (que basicamente transformar um dado de um formato para outro) de uma URL. De certa forma, uma biblioteca que realiza o request de uma URL, que possibilita a extrao de dados feitas pelo BeautifulSoup.

requests

Definio: uma biblioteca que requisita o acesso a uma URL. De certa forma, a biblioteca requests considerada como easy-to-use ao ser comparada com a urlib. Apesar disso, a urllib uma biblioteca que apresenta algumas funes a mais que a requests no apresenta.

BeautifulSoup (bs4)

Definio: uma biblioteca para extrair dados de HTML e arquivos XML. Neste sentido, o bs4 uma biblioteca que necessita de bibliotecas como a urlib e a requests para poder funcionar. No geral, tem timos resultados e funciona de forma eficiente.

def get_func(module):

    exec('import scrappers.' + module, globals())
    exec('funcs = getmembers(scrappers.' + module + ', isfunction)', globals())

    return [f[0] for f in funcs if module.lower() in f[0].lower()]

A varivel name_scrappers uma lista com o nome de todos os arquivos que esto contidos na pasta scrappers do diretrio do projeto. A funo get_func() recebe o nome de um desses arquivos e retorna as funes de scrapping que aquele arquivo possui. Um exemplo de chamada desta funo com o mdulo speciesconservation retorna o seguinte resultado: get_func('speciesconservation') = {{get_func('speciesconservation')}}. def call_fun(fun, path):

    global keywords
    # print(f'Incio de {fun}') # esta linha de cdigo foi comentada para deixar o relatrio mais sucinto,  recomendvel
    # deixar ela ativa para saber exatamente qual funo que deu problema.
    if f[-1] != '1':
        text = fun + f"('{path}')"
        text = text.replace("\\", "\\\\")
    else:
        text = fun + f"('{path}', '{keywords}')"
        text = text.replace("\\", "\\\\")
    return eval(text)

def atualizador(baseprincipal, diamaisrecente):
    diario = pd.read_csv(diamaisrecente)
    main = pd.read_csv(baseprincipal)
    # checando o que do 'diario' est no 'main'
    a = diario['link'].isin(main['link'])  # usar o tag/id criado inves do link
    b = [not bool
         for bool in a]  # inverter para o TRUE ser a linha que no tem no main
    novaslinhas = diario[b]
    main = main.append(novaslinhas, ignore_index=True)  # novo main
    main.to_csv(baseprincipal, index=False, sep=",")
    #print('A base foi atualizada')
